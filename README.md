# eslint-plugin-oxygen
My own eslint plugin, used by my own project.

# usage

```shell
npm install eslint-plugin-oxygen -D
```

```javascript
// .eslintrc.cjs
module.exports = {
  plugins: ['oxygen'],
  extends: ['plugin:oxygen/react']
}
```

- eslint
- eslint-plugin-react
- @typescript-eslint/eslint-plugin
- eslint-plugin-html
